# Roadmap

- Stable websocket chat with private rooms and permissions
- JWT auth with refresh tokens
- QuizBot with progressive hints and leaderboard
- Dockerized deployment

- Moderation dashboard
- Message search and pinned messages

## v1.2.0

- Advanced anti-abuse controls
